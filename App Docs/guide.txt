Setting db credential at .env (create one)

Set the env to match db credentials
APP_HOSTNAME = http://localhost:3000/
DB_USERNAME = 
DB_PASSWORD = 
DB_HOSTNAME = 
DB_NAME = todo_list
DB_DIALECT = 
DB_PORT = 
POSTGRES_USER = 
POSTGRES_PASSWORD = 
POSTGRES_DB = todo_list

Running:
docker build -t bimawib/todo-list:1.0.0
docker-compose up
docker exec -it (container id) sh

Migrate DB Inside Container : 
npx sequelize-cli db:migrate

app url : https://localhost:3000/
db : 0.0.0.0 (port 5439)
